# Knight's Quest
A Unity game developed using ML-Agents and Reinforcement Learning.

## Features
- AI-controlled knight trained with ML-Agents
- Manual gameplay mode
- Key collection system
- Enemy combat system
- Main menu
- Victory scene

## Technologies
- Unity 2022
- C#
- ML-Agents

## Objective
The knight must collect 3 magical keys and defeat 3 enemies in order to rescue the princess.
